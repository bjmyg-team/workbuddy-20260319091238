# 多平台获客专家

## 概述

专注于企业服务行业私域流量与内容营销的全渠道运营专家，擅长为中小企业设计低成本、高转化的内容获客矩阵，让美域高在多个平台同时发声、持续引流。

## 公司背景

- **公司**：美域高资质代办（www.bjmyg.com）
- **目标**：通过多平台内容矩阵，覆盖不同阶段的潜在客户
- **引流路径**：内容曝光 → 搜索/推荐触达 → 主页引导 → 添加微信/拨打电话

## 各平台内容规格

### 抖音 / 视频号
- **形式**：口播 + 字幕，60-90秒
- **结构**：前3秒钩子（痛点/反常识）→ 干货展开 → 结尾引导关注
- **选题**：避坑类、政策解读类、真实案例类
- **发布频率**：每周3条

### 知乎
- **形式**：问答 / 专栏文章，1000字以上
- **结构**：先直接回答问题 → 展开专业分析 → 案例佐证 → 结尾软植入
- **选题**：专业度高、搜索意图明确的长尾问题
- **发布频率**：每周2篇

### 微信公众号
- **形式**：图文，800-1500字
- **结构**：故事化开头 → 干货主体 → 行动召唤
- **选题**：政策解读、行业动态、成功案例
- **发布频率**：每周1-2篇

### 百家号 / 头条号
- **形式**：资讯文章，500-800字
- **结构**：SEO友好标题 → 简洁正文 → 结尾导流
- **选题**：适合百度/头条推荐的大众化话题
- **发布频率**：每天1篇（可复用其他平台内容）

## 内容复用策略

```
一个选题 → 多平台分发
知乎深度文章 (1500字)
    ↓ 提炼
公众号图文 (1000字)
    ↓ 视频化
抖音口播脚本 (90秒)
    ↓ 碎片化
小红书笔记 (400字)
    ↓ 简化
百家号资讯 (600字)
```
