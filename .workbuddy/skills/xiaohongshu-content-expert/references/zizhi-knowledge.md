# 企业资质知识库

## ICP 增值电信业务许可证

**适用对象**：开展互联网信息服务、电子商务、网络游戏等业务的企业

**申办条件**：
- 注册资本：不低于100万元人民币
- 有固定的办公场所
- 有相应的专业技术人员
- 有健全的网络与信息安全保障措施

**办理流程**：
1. 准备材料（营业执照、法人身份证、场地证明等）
2. 在工业和信息化部或省级通信管理局官网提交申请
3. 等待审核（30-60个工作日）
4. 领取许可证

**常见问题**：
- ICP备案 ≠ ICP许可证（备案免费，许可证需要资质审查）
- 跨省经营需在工信部申请，省内经营在省级管局申请

## 食品经营许可证

**适用对象**：餐厅、外卖、食品销售、食品加工等

**申办条件**：
- 有与经营品种相适应的设备设施
- 有合理的设备布局和工艺流程
- 有专职或兼职的食品安全管理人员

**办理流程**：
1. 准备材料（营业执照、场地租赁合同、平面图等）
2. 向所在地市场监督管理局提交申请
3. 现场核查（重要！要提前准备）
4. 审批发证（20个工作日内）

**办理周期**：通常15-30个工作日

## 高新技术企业认定

**核心价值**：企业所得税从25%降至15%，每年可节省大量税款

**认定条件**：
- 注册满一年
- 拥有自主知识产权（专利/软件著作权）
- 研发费用占收入比例达标（小企业≥5%）
- 高新技术产品收入占总收入≥60%
- 科技人员占员工总数≥10%

**申报周期**：每年一次，一般3-9月申报

**有效期**：3年，到期可重新认定

## ISO 9001 质量管理体系认证

**适用场景**：招投标加分、提升企业管理水平、出口贸易需要

**办理流程**：
1. 申请（选择认证机构）
2. 体系建立（建立文件和流程，约1-3个月）
3. 内审
4. 管理评审
5. 认证审核（初审+正式审核）
6. 颁发证书

**有效期**：3年，每年监督审核一次

## 人力资源服务许可证

**适用对象**：劳务派遣公司、人才中介、猎头公司

**申办条件**：
- 注册资本200万元以上（劳务派遣）
- 有固定的经营场所
- 有相应的专业人员

**办理部门**：人力资源和社会保障局
