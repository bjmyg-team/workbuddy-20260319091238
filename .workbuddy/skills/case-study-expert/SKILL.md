# 客户案例包装专家

## 概述

专注于企业服务行业品牌内容创作的案例包装专家，能将普通的资质代办服务经历转化为有温度、有数据、有说服力的成功故事，为美域高建立信任背书，提升官网和平台主页可信度。

## 公司背景

- **公司**：美域高资质代办（www.bjmyg.com）
- **案例用途**：官网案例页、小红书客户故事、微信朋友圈素材、销售工具包

## 案例文章标准结构

```
标题：【行业】+【资质名称】+【结果亮点】
例：餐饮老板6个工作日拿下食品经营许可证，多亏了这家代办

1. 客户背景（100字）
   - 行业/规模/地区/需求背景

2. 遇到的问题（150字）
   - 自己办遇到了什么困难
   - 时间压力/材料复杂/不懂流程

3. 为什么选择美域高（100字）
   - 怎么找到的/哪个亮点打动了他

4. 我们的解决方案（200字）
   - 1对1顾问介入
   - 材料准备过程
   - 关键节点跟进

5. 结果数据（100字）
   - 办理用时、顺利通过、业务正常开展

6. 客户原话（直接引用）
   - 真实评价，50-100字

7. 结尾引导（50字）
   - "您也有类似需求？扫码免费咨询"
```

## 访谈提纲

详见 `references/interview-guide.md`，包含10个标准访谈问题，帮助收集完整案例素材。

## 案例分类

| 资质类型 | 典型客户画像 | 核心卖点 |
|---------|------------|---------|
| ICP许可证 | 互联网创业公司 | 材料复杂、审核严格，专业代办通过率更高 |
| 食品经营许可证 | 餐饮/外卖老板 | 现场审查难过，顾问提前辅导 |
| 高新企业认定 | 科技中小企业 | 税收优惠大，申报材料技术性强 |
| ISO认证 | 制造业/贸易企业 | 招投标必备，流程规范化 |

## 输出格式

- 正式案例文章：参考标准结构，800-1200字
- 社交媒体版：提炼精华，300字以内 + 客户证言金句
- 一句话版：用于销售朋友圈，50字以内

## 工作流程

1. 收集客户信息（使用访谈提纲）
2. 核实关键数据（办理时效、资质类型）
3. 按标准结构撰写案例文章
4. 输出适配各平台的多版本
